# 欢迎来到我的博客 🎉

欢迎来到我的全新博客！这是一个基于 **Next.js 14** 构建的现代化博客系统，专注于提供优质的阅读体验和便捷的内容管理。

## 🚀 主要特性

### 1. 强大的技术栈

- **Next.js 14+** - 使用最新的 App Router 架构
- **TypeScript** - 完整的类型安全
- **Tailwind CSS** - 现代化的样式系统
- **Markdown/MDX** - 简单的内容创作方式

### 2. 优秀的用户体验

- ✨ **响应式设计** - 完美适配各种设备
- 🌓 **暗色模式** - 护眼的深色主题
- 🔍 **全文搜索** - 快速找到你想要的内容
- 📱 **移动端优化** - 流畅的移动端体验

### 3. 丰富的功能

博客系统包含了以下功能：

- 文章列表和详情展示
- 标签和分类系统
- 文章目录（TOC）
- 阅读时间估算
- 上一篇/下一篇导航
- RSS 订阅支持
- SEO 优化

## 💻 代码高亮

支持多种编程语言的语法高亮：

```javascript
// JavaScript 示例
function greet(name) {
  console.log(`Hello, ${name}!`);
}

greet('World');
```

```python
# Python 示例
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

print(fibonacci(10))
```

```typescript
// TypeScript 示例
interface User {
  id: number;
  name: string;
  email: string;
}

const user: User = {
  id: 1,
  name: 'John Doe',
  email: 'john@example.com'
};
```

## 📝 Markdown 支持

支持完整的 Markdown 语法：

**无序列表：**

- 第一项
- 第二项

  - 嵌套项 2

**有序列表：**

3. 最后

## 🎨 设计理念

这个博客系统的设计遵循以下原则：

1. **简洁优雅** - 清晰的视觉层次，舒适的阅读体验
2. **性能优先** - 静态生成，快速加载
3. **易于维护** - 简单的 Markdown 文件管理
4. **开发者友好** - 清晰的代码结构，完整的注释

## 🚀 开始使用

只需在 `content/posts/` 目录下创建 Markdown 文件，就可以开始写作了！

每篇文章都支持 Front Matter 元数据：

```markdown
---
title: "文章标题"
date: "2025-11-23"
tags: ["标签1", "标签2"]
category: "分类"
description: "文章描述"
---
```
